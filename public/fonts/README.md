Place licensed Neue Montreal font files in this directory:

- `NeueMontreal-Regular.woff2`
- `NeueMontreal-Regular.woff`
- `NeueMontreal-Medium.woff2`
- `NeueMontreal-Medium.woff`

The site already includes `@font-face` rules for these filenames in `styles/globals.css`.
If the files are not present, the UI falls back to DM Sans.
